# 🌍 AI Multi-Language Translator App

An AI-powered translation web application that converts English text into multiple languages like Spanish, French, and Japanese. The app works similar to a messaging interface, providing real-time translations using the OpenRouter API.

---

## 🚀 Features

- 🌐 Translate English text into:
  - Spanish 🇪🇸
  - French 🇫🇷
  - Japanese 🇯🇵
- 💬 Messenger-style chat interface
- 🤖 AI-powered translations
- ⚡ Real-time responses
- 📋 Copy translated text easily
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- **Frontend:** React.js / Next.js  
- **Backend:** Node.js (optional)  
- **API:** OpenRouter (AI-based translation)  
- **Styling:** Tailwind CSS / CSS  

---

## 🔑 How It Works

1. User enters text in English  
2. The app sends a request to the OpenRouter API  
3. AI processes the input and translates it  
4. Translated text is displayed in selected languages
